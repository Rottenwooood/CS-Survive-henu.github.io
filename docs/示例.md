# 示例

> 每周总结一次，整理一下发到这里。（一般来说是知乎的文章而不是书）

## [深入理解计算机系统](https://csapp.cs.cmu.edu/)
**摘要**: 这本书从程序员的角度深入理解计算机系统，涵盖程序如何表示和执行、数据如何存储、程序如何交互等内容。适合想要深入理解计算机工作原理的同学。

## [算法导论](https://mitpress.mit.edu/9780262046305/introduction-to-algorithms/)
**摘要**: MIT 的经典算法教材，全面介绍了算法的设计与分析。内容深入浅出，从基础排序到高级数据结构，适合系统学习算法知识。
